Data Viz

An analysis and visualization of some data points collected from the class.

Objectives

  • Navigate arrays.
  • Work with data from arrays.
  • Visualize data from JavaScript as HTML elements and CSS properties.

Procedure

  1. Find the Week 7 assignment in your Team (class section) on Replit and click "Start project" to begin.
  2. You and your classmates were recently polled on the following:
    • How far is your hometown from Miami University?
    • How many pets are currently in your family?
  3. Design visualizations for this data that do the following:
    • Show the average distance that your classmates collectively travel to go home (text).
    • Show the total distance that your classmates collectively travel to go home (text).
    • Show how many of each type of pet are in your classmates' families (graphic comparison).
    • Show the total number of all pets in your classmates' families (text).
  4. The data from the responses to these questions will be provided for you as arrays in the assignment template. Although this is a fairly small dataset that would be trivial to calculate by hand, you MUST do all data analysis using JavaScript i.e. all values should be collected and calculated using the array length properties and the values in the arrays.
  5. You will need to use the results that you calculate in JavaScript to directly change HTML attributes and/or CSS properties. None of these should be hard-coded into your HTML or CSS - they will be updated by the JavaScript after the page initi ally loads. Think about some of the ways you can approach this and the decisions you will need to make:
    1. How will you calculate averages and totals where needed?
    2. What kind of graph(ic) would be most appropriate for the pet type breakdown?
    3. How can you construct this graph(ic) using just HTML and CSS?
    4. How can you visualize proportions using CSS units e.g. px, %, flex units?
    5. What other attribute/properties might you need to change to show the output from your JavaScript analysis?
  6. The overall styling and presentation of this visualization should be fairly polished with consideration for color, fonts, layout, and placement on the page.
  7. Remember to click the green "Run" button as needed to refresh the browser preview update on the right side of the window.

You may click "Resubmit" as many times as you'd like to submit changes/updates - however, keep in mind that this will also change the timestamp on your submission.
